# CoTabor 飞书设置手册

为了让 AI 拥有记忆与日志能力，CoTabor 支持将飞书多维表格作为后台数据库。按照以下步骤，两分钟即可完成飞书后端的连接与初始化。

## 第一步：飞书扫码授权

1. 打开 CoTabor 插件的选项设置页（点击插件图标，在下拉面板点击 ⚙️ 设置按钮）。
2. 在左侧菜单栏选择 **飞书设置**。
3. 点击 **扫码登录飞书** 按钮。
4. 此时会打开一个新的飞书授权网页，请使用飞书手机端扫描页面上的二维码，并确认授权给 CoTabor 插件。
5. 授权成功后页面将自动关闭，设置界面会显示已登录以及你的飞书用户名。

> *提示：CoTabor 会获取并使用合规的飞书官方授权 Token 进行接口调用，安全可靠，不接触任何其它个人信息。*

## 第二步：构建 AI 记忆仓库

由于底层调用方式，我们需要你创建一个用来存放配置与数据的空文件夹：
1. 打开飞书云文档空间：[飞书云文档](https://feishu.cn/drive/)。
2. 在适合的地方（比如“我的空间”根目录）创建一个 **新的空文件夹**，并命名为类似 "CoTabor_Memories"。
3. 复制该文件夹的完整网页链接（类似 `https://xxx.feishu.cn/drive/folder/xxx...`）。
4. 切回 CoTabor 设置页，在“步骤 2：构建多维表格”将刚才得到的 **文件夹链接（或 Folder Token）粘贴到输入框中**。
5. 点击 **一键初始化 AI 数据中心**。

> 稍等几秒钟，系统将在你指定的文件夹下自动创建专门记录运行日志与学习记忆的多维表格数据库底座。当界面提示“初始化成功！”时，飞书后端便已彻底打通。

**完毕！** 现在 CoTabor 就可以自动记录上下文，积累属于个人的业务操作规则与偏好了。
